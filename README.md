# Digital Safety Toolkit

Digital Safety Toolkit is a lightweight full-stack prototype designed to identify potentially harmful or misleading digital content and improve user-facing digital safety awareness.

## Overview
This project focuses on analyzing input text and detecting potential risks such as spam, suspicious links, and misleading content using rule-based logic.

## Features
- Detects spam and suspicious keywords
- Identifies potentially misleading or unsafe content
- Flags presence of external links
- Classifies input into risk levels (Low / Medium / High)
- Provides explanation-based output for better user understanding

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Python Flask  

## How It Works
- User inputs text
- Backend analyzes patterns using rule-based logic
- System assigns a risk level
- Output includes detected issues and explanation

## Purpose
This project was built as an early-stage trust-and-safety prototype to explore practical problems in:
- Digital safety  
- Misinformation detection  
- User-aware software systems

- Improve detection accuracy
